VIDEO PRODUCTION

PHP / Laravel 12.1.1

Project Description:

Web App running in 2019 on Laravel 5.7.14 / Php 7.4.33 (or prev).
Totally re-built on Laravel 12.1.1 / Php 8.2.0.
Added differents fixes to work with new Laravel version due to code not valid anymore on old Laravel.

Migration at beginning have been tried but the code written after upgrade of Php/Laravel was not working. It was therefore necessary to rewrite the application, solving the problems created by the code of the old version.
